# Local Development Notes

## Shell Escaping
- `node -e` with `!==` fails in zsh (history expansion). Write a temp `.mjs` file instead.
- `@tonejs/midi` is CJS — use `import pkg from '@tonejs/midi'; const { Midi } = pkg;` in ESM scripts.
- `@tonejs/midi` is only in `packages/midi/node_modules` (pnpm hoisting). Run scripts from that directory.
