# MIDI Visualization References

Research notes for piano roll and MIDI visualization inspiration.

## Web-Based (try in browser)

- **[Signal](https://signalmidi.app/)** — Full multitrack MIDI editor in the browser. Open source ([GitHub](https://github.com/ryohey/signal)), MIT license, React-based, GM sound module with 128 instruments. Most architecturally relevant — solves similar multitrack piano roll UX problems.

- **[Midiano](https://midiano.com/)** — Falling notes over a piano with GPU-accelerated particle engine. Beautiful visual polish, good WebGL rendering reference.

- **[MIDI Toolbox Visualizer](https://miditoolbox.com/visualizer)** — Clean piano roll viewer with zoom, track filtering, and note inspection. Runs locally in browser.

- **[Pianotify](https://pianotify.com/online-piano-roll)** — Piano roll videos with beat-aligned grid, per-track color/mute controls, virtual keyboard with real-time key highlighting.

- **[PianoVisual](https://pianovisual.com/)** — Simple browser player with theme options and real-time note highlighting.

## Open Source (GitHub)

- **[pixi-piano-roll](https://github.com/mjhasbach/pixi-piano-roll)** — WebGL/Canvas animated piano roll using PixiJS. Good rendering approach reference.

- **[Magenta.js Visualizer](https://magenta.github.io/magenta-js/music/demos/visualizer.html)** — Google's music ML library includes a clean MIDI visualizer component.

- **[adamjmurray/pianoroll](https://github.com/adamjmurray/pianoroll)** — HTML5 Piano Roll: MIDI visualizer, player, and editor.

- **[Euphony](https://github.com/qiao/euphony)** — WebGL MIDI visualizer.

## Desktop (polished references)

- **[SeeMusic](https://www.seemusicapp.com/)** — 3D falling notes with particles and lighting. Sets the visual bar for MIDI visualization.

- **[MIDIVisualizer](https://github.com/kosua20/MIDIVisualizer)** — Cross-platform OpenGL visualizer, very popular. Rhythm game aesthetic.

- **[Neothesia](https://polymeilex.github.io/Neothesia/)** — Cross-platform falling notes visualizer, Synthesia-style.
